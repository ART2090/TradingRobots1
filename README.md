# TradingRobots1
A trading robot of hyperliquid

A high-performance cryptocurrency quantitative trading system written in Go.
Supports live trading, backtesting, multiple exchanges, and modular strategies.

## Features
- High-performance Go architecture
- Modular strategy engine
- Backtesting framework
- Exchange abstraction layer
- Risk management module
- SQLite3 storage
- Configurable YAML settings
